Sub Domain
=================================

#. Expand "Forward Lookup Zones", and right click on "F5DEMO.com

   .. image:: /_static/class7/dc01_new_delegation.png

#. Create the "wip" subdomain.

   .. image:: /_static/class7/dc01_new_delegation_create_gslb.png

#. Step through the Delegation Wizard. Add "ns1.f5demo.com - 10.1.10.101"

   .. image:: /_static/class7/dc01_new_delegation_ns1ns2.png

#. Also add "ns2.f5demo.com - 10.1.60.101"

   .. image:: /_static/class7/dc01_new_delegation_ns1ns2_create.png

#. Make sure both ns1.f5demo.com and ns2.f5demo.com are added

   .. image:: /_static/class7/dc01_new_delegation_ns1ns2_create_finish.png

#. Click "Finish"

   .. image:: /_static/class7/dc01_new_delegation_create_gslb_finish.png
